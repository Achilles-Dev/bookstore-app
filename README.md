# Bookstore App

> This apllication contains a list of books with their completion stages.


## Built With

- JavaScript
- React


## Getting Started

To get a local copy up and running follow these simple example steps.

1. Clone Repository 
  
  Run
   ```
   git clone git@github.com:Achilles-Dev/react-todo-app.git
   ```

### Install

  Run
   ```
   npm install 
   ```
